정보를, 필요할 때 바로 찾아볼 수 있는 서비스를 제공 하려 합니다

			@Override
			public Long register(CustomerQuestionDTO customerQuestionDTO) {
				Map entityMap = dtoToEntity(customerQuestionDTO);
		
				CustomerQuestion customerQuestion = (CustomerQuestion) entityMap.get("customerQuestion");
				customerQuestionRepository.save(customerQuestion);
				if (entityMap.get("customerImages") != null) {
					List customerImages = (List) entityMap.get("customerImages");
					customerImages.forEach(images -> customerImageRepository.save(images));
				}
		
				return customerQuestion.getCono();
			}

			@Override
    @Transactional
    public Long register(AdminReplyDTO adminReplyDTO) {
        AdminReply adminReply = adminReplyRepository.save(dtoToEntity(adminReplyDTO));

        CustomerQuestion customerQuestion = customerQuestionRepository
                .findById(adminReply.getCustomerQuestion().getCono()).get();
        customerQuestion.setClear(true);
        customerQuestionRepository.save(customerQuestion);

        return adminReply.getArno();
    }

이전 코드와 비교


  • 간편해진 유지보수
  • 프로젝트 용량 하락
  • 메모리 소비 최소화로 성능 증가

자체 평가 및 느낀점


  • - 시간을 들여 프로젝트의 전체적인 구상과 설계를 팀원과의 회의를 통해 다져 놓았기에 업무가 분담 되었더라도 각 팀원간의 정보전달 및 연동이 매우 빠르게 진행되었습니다.

  • - 전반적으로 짜여진 계획 및 구상을 토대로, 데이터 베이스 설계를 한 명이 전담하여 구체적으로 작성한 뒤 배포, 다른 팀원들은 설계가 진행되는 동안 각자의 디자인, 페이지 구상 등을 계획하여 시간을 효율적으로 사용 할 수 있었습니다.

  • - 하지만 페이지 자체의 디테일적인 파트별 기능 및 디자인을 의논을 통해 확실하게 구상하지 않았던 것이 추후 작업에 들어가는 시간에 많은 지장을 주었습니다.

  • - 또한 하나의 팀으로서의 작업, 유지보수적 측면에서 각 코드의 최적화가 프로젝트에 주는 영향이 어떤지 알 수 있었습니다

					Member member = null;

					}
			
					pageRequestDto.setSize(16);
					log.info("전체 gallery service list 요청 {}", pageRequestDto);
			
					Page result = galleryRepository.getTotalList(pageRequestDto.getType(),
							pageRequestDto.getKeyword(),
							myPet,
							member,
							pageRequestDto.getPageable(Sort.by("gno").descending()));
			
					Function function = null;
			
					if (memberDto != null) {
						function = (en -> entityToDto((Gallery) en[0],
								(List) Arrays.asList((GalleryImage) en[1]), (Long) en[2]));
					} else {
						function = (en -> entityToDto((Gallery) en[0],
								(List) Arrays.asList((GalleryImage) en[1]), null));
					}
			
					return new PageResultDTO<>(result, function, 10);
				}
