팀장 유창규

프로젝트 개요


반려동물을 키우는데 있어, 처음에 무엇을 준비해야하는지
상식적으로 알고있어야하는 정보는 무엇인지,

검색을 하더라도 원하는 정보는 여기저기 흩어져 숨어있기라도 하듯
찾아다녀야하는 불편함을 해소하기위해
반려동물을 위한 정보를 간편하게 한눈에 들어오게끔 제공하는것이 목적

쉬운 사용자 접근성
빠른 정보전달
필요시 즉각적인 정보 확인

담당 업무


  • ○ 베이스 레이아웃 및 메인 페이지 구성, 고객센터 구축

  • ○ 전체적인 팀원들의 에러 수정 및 방향 제시

  • ○ 프로젝트 Github 설정 및 관리

  • ○ 데이터베이스 기반 설계 후 배포

기본 베이스 템플릿 제공

  • Index
  • Layout
  • Fragment
    - Header     , Footer
